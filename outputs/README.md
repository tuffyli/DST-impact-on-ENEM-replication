# Outputs

This folder stores tables, figures, maps, and auxiliary exports generated by the
analysis scripts.

Expected structure:

```text
outputs/
  controls/
    figures/
      pdf/
    maps/
  scores/
    figures/
      pdf/
  migration/
  stata/
```

## Contents

`outputs/controls/`
Stores most LaTeX regression tables.

`outputs/controls/figures/`
Stores main regression figures and robustness figures in image format.

`outputs/controls/figures/pdf/`
Stores PDF or EPS versions of figures for LaTeX inclusion.

`outputs/controls/maps/`
Stores maps produced by `05_maps.R`.

`outputs/scores/`
Stores score-distribution and balance-test outputs.

`outputs/migration/`
Stores migration and sample-composition tables.

`outputs/stata/`
Stores Stata exports used for external figure or table workflows.

## Reproducibility Rule

Files in this folder are generated products. They can be deleted and recreated
by rerunning the relevant scripts from `01_master.R`.
